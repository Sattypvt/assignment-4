# 🎨 AI Image Studio

An AI-powered image generation web application built with **Python** and **Streamlit** using the **Pollinations AI API**. Users can generate high-quality AI images from text prompts, customize image dimensions, enhance prompts with AI, and download generated images.



## ✨ Features

- 🖼️ Generate AI images from text prompts
- 🎨 Multiple Art Styles:
  - Realistic
  - Anime
  - Fantasy
  - Cyberpunk
  - Oil Painting
  - Watercolor
- 📏 Adjustable image width and height
- ✨ Magic Enhance feature for richer AI-generated images
- 🎲 Surprise Me button for random creative prompts
- 📥 Download generated images as PNG files with dynamic filenames
- 📱 Clean and interactive Streamlit user interface

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Sattypvt/ai-image-studio.git
```

### 2. Navigate to the project folder

bash
cd ai-image-studio
```

### 3. Install dependencies

```bash
pip install -r requirements.txt


### 4. Run the application

```bash
streamlit run app.py
```
